# 🌅 VIT Bhopal Morning Briefing Bot

## 📖 About The Project
The **VIT Bhopal Morning Briefing Bot** is a Python-based command-line tool designed to help students start their day efficiently. Instead of checking multiple apps for weather, news, commute times, and daily motivation, this bot aggregates everything into one clean, time-stamped terminal dashboard. 

This project was built as a capstone "Bring Your Own Project" (BYOP) to solve the real-world problem of morning smartphone distractions and information overload.

### ✨ Features
* **🌤️ Live Weather:** Fetches real-time temperature and wind speed for the VIT Bhopal (Kothri Kalan) campus using the Open-Meteo API.
* **📰 Dynamic Local News:** Pulls a randomized selection of 5 recent news headlines related to "Bhopal" using the NewsAPI.
* **🗺️ Smart Commute Logic:** Simulates live traffic conditions and travel times for routes to Bhopal City and Indore City, recommending the best conditions.
* **🌟 Daily Inspiration:** Retrieves a randomized inspirational quote from the ZenQuotes API to start the day right.
